📚 Class 10 Chemistry - Complete Formula Reference

🧪 Acids, Bases & Salts

Common Acids
HCl - Hydrochloric Acid
H2SO4 - Sulphuric Acid
HNO3 - Nitric Acid
CH3COOH - Acetic Acid
Example: HCl is present in gastric juice
Common Bases
NaOH - Sodium Hydroxide
KOH - Potassium Hydroxide
Ca(OH)2 - Calcium Hydroxide
Mg(OH)2 - Magnesium Hydroxide
NH4OH - Ammonium Hydroxide
Example: Ca(OH)₂ is used in whitewashing
Neutralization Reaction
Acid + Base → Salt + Water
HCl + NaOH → NaCl + H2O
Example: When antacid neutralizes stomach acid
Acid + Metal Reaction
Acid + Metal → Salt + Hydrogen
2HCl + Zn → ZnCl2 + H2↑
Example: Zinc reacts with hydrochloric acid
pH Scale
pH = -log[H+]
pH < 7 = Acidic
pH = 7 = Neutral
pH > 7 = Basic
Example: Lemon juice pH ≈ 2, Water pH = 7

⚙️ Metals and Non-metals

Metal + Oxygen
Metal + Oxygen → Metal Oxide
4Na + O2 → 2Na2O
Example: Sodium burns in oxygen
Metal + Water
Metal + Water → Metal Hydroxide + Hydrogen
2Na + 2H2O → 2NaOH + H2↑
Example: Sodium reacts vigorously with water
Metal + Acid
Metal + Acid → Salt + Hydrogen
Mg + 2HCl → MgCl2 + H2↑
Example: Magnesium ribbon in HCl
Displacement Reactions
More reactive metal + Less reactive metal salt → Less reactive metal + More reactive metal salt
Zn + CuSO4 → ZnSO4 + Cu
Example: Zinc displaces copper from copper sulphate
Extraction of Metals
Metal Oxide + Carbon → Metal + Carbon Dioxide
ZnO + C → Zn + CO
2PbO + C → 2Pb + CO2
Example: Reduction of zinc oxide
Corrosion of Iron
4Fe + 3O2 + 6H2O → 4Fe(OH)3 (Rust)
Example: Rusting of iron in presence of oxygen and water

📊 Periodic Classification of Elements

Atomic Number
Z = Number of protons = Number of electrons (in neutral atom)
Example: Hydrogen Z = 1, Carbon Z = 6
Mass Number
A = Number of protons + Number of neutrons
Example: Carbon-12 has mass number 12
Electronic Configuration
Maximum electrons in shell = 2n2
K shell (n=1): 2 electrons
L shell (n=2): 8 electrons
M shell (n=3): 18 electrons
Example: Sodium (11): 2, 8, 1
Valency
Valency = 8 - Number of electrons in outermost shell (if >4)
Valency = Number of electrons in outermost shell (if ≤4)
Example: Chlorine (2,8,7) has valency 1
Modern Periodic Law
Properties of elements are periodic functions of their atomic numbers
Example: Elements in same group have similar properties

🌱 Life Processes - Chemical Formulas

Photosynthesis
6CO2 + 6H2O + Light Energy → C6H12O6 + 6O2
Example: Plants make glucose using carbon dioxide and water
Respiration
C6H12O6 + 6O2 → 6CO2 + 6H2O + Energy (ATP)
Example: Glucose is broken down to release energy
Anaerobic Respiration (Fermentation)
C6H12O6 → 2C2H5OH + 2CO2 + Energy
Example: Yeast converts glucose to ethanol
Water Formula
H2O - Water
Example: Essential for all life processes
Important Gases
O2 - Oxygen
CO2 - Carbon Dioxide
N2 - Nitrogen
Example: Oxygen is needed for respiration
