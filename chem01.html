Chemical Reactions and Equations

Chemical Reaction
  • Process where reactants transform into products.
  • Characteristics: Change in color, state, temperature, gas evolution, precipitate formation.
  • Examples:
    • Burning of magnesium: 2Mg + O₂ → 2MgO
    • Rusting of iron: 4Fe + 3O₂ + 2xH₂O → 2Fe₂O₃·xH₂O
    • Reaction of vinegar and baking soda: CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂
Chemical Equation
  • Symbolic representation of a chemical reaction.
  • Reactants (left) → Products (right).
  • Examples:
    • Water formation: 2H₂ + O₂ → 2H₂O
    • Ammonia synthesis: N₂ + 3H₂ → 2NH₃
    • Photosynthesis: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
Balanced Chemical Equation
  • Equal number of atoms of each element on both sides.
  • Law of conservation of mass obeyed.
  • Steps: Write unbalanced equation, balance atoms, adjust coefficients.
  • Examples:
    • Methane combustion: CH₄ + 2O₂ → CO₂ + 2H₂O
    • Aluminum reaction with oxygen: 4Al + 3O₂ → 2Al₂O₃
    • Potassium chlorate decomposition: 2KClO₃ → 2KCl + 3O₂
Types of Chemical Reactions
  • Combination: Two or more substances form one product (A + B → AB).
    • CaO + H₂O → Ca(OH)₂
    • 2H₂ + O₂ → 2H₂O
    • C + O₂ → CO₂
  • Decomposition: Single compound breaks into two or more products (AB → A + B).
    • 2H₂O → 2H₂ + O₂
    • CaCO₃ → CaO + CO₂
    • 2HgO → 2Hg + O₂
  • Displacement: More reactive element displaces less reactive element (A + BC → AC + B).
    • Zn + CuSO₄ → ZnSO₄ + Cu
    • Fe + CuSO₄ → FeSO₄ + Cu
    • Cl₂ + 2NaBr → 2NaCl + Br₂
  • Double Displacement: Exchange of ions between two compounds (AB + CD → AD + CB).
    • NaCl + AgNO₃ → AgCl + NaNO₃
    • BaCl₂ + Na₂SO₄ → BaSO₄ + 2NaCl
    • Pb(NO₃)₂ + 2KI → PbI₂ + 2KNO₃
  • Combustion: Substance reacts with oxygen, releasing energy.
    • C₃H₈ + 5O₂ → 3CO₂ + 4H₂O
    • C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O
    • CH₄ + 2O₂ → CO₂ + 2H₂O
Exothermic and Endothermic Reactions
  • Exothermic: Release heat, temperature rises.
    • Burning of coal: C + O₂ → CO₂ + heat
    • Neutralization: HCl + NaOH → NaCl + H₂O + heat
    • Respiration: C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + energy
  • Endothermic: Absorb heat, temperature drops.
    • Decomposition of CaCO₃: CaCO₃ → CaO + CO₂
    • Photosynthesis: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
    • Ammonium chloride dissolution: NH₄Cl + H₂O → NH₄⁺ + Cl⁻
Redox Reactions
  • Involve reduction (gain of electrons) and oxidation (loss of electrons).
  • Oxidizing agent: Accepts electrons. Reducing agent: Donates electrons.
  • Examples:
    • 2Mg + O₂ → 2MgO (Mg oxidized, O₂ reduced)
    • CuO + H₂ → Cu + H₂O (CuO reduced, H₂ oxidized)
    • Zn + 2HCl → ZnCl₂ + H₂ (Zn oxidized, H⁺ reduced)
Corrosion
  • Gradual deterioration of metals by chemical reaction with environment.
  • Prevention: Painting, galvanizing, alloying.
  • Examples:
    • Rusting of iron: 4Fe + 3O₂ + 2xH₂O → 2Fe₂O₃·xH₂O
    • Corrosion of copper: 2Cu + O₂ + CO₂ + H₂O → Cu₂(OH)₂CO₃ (green patina)
    • Corrosion of silver: 2Ag + H₂S → Ag₂S + H₂ (tarnishing)
Rancidity
  • Oxidation of fats/oils in food, causing bad smell/taste.
  • Prevention: Antioxidants, refrigeration, vacuum packing.
  • Examples:
    • Spoilage of butter (oxidation of fats)
    • Rancid oil in potato chips
    • Stale nuts due to lipid oxidation
Chemical Equation Symbols
  • (s): Solid, (l): Liquid, (g): Gas, (aq): Aqueous solution.
  • →: Yields, ⇌: Reversible reaction.
  • ↑: Gas evolved, ↓: Precipitate formed.
  • Examples:
    • CaCO₃(s) → CaO(s) + CO₂(g)↑
    • AgNO₃(aq) + NaCl(aq) → AgCl(s)↓ + NaNO₃(aq)
    • 2Na(s) + Cl₂(g) → 2NaCl(s)
Factors Affecting Reaction Rate
  • Concentration: Higher concentration increases rate.
  • Temperature: Higher temperature increases rate.
  • Catalyst: Lowers activation energy, speeds up reaction.
  • Surface area: Smaller particles increase rate.
  • Examples:
    • Higher concentration of HCl speeds up Zn + 2HCl → ZnCl₂ + H₂
    • Heating CaCO₃ accelerates decomposition: CaCO₃ → CaO + CO₂
    • Catalyst in Haber process (N₂ + 3H₂ → 2NH₃) increases ammonia production
